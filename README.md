# Blue Innovation. Bespoke Landing Page.

This page is a single-prospect landing surface built for Blue Innovation Co., Ltd. (TSE:5597) to be shown at eMerge Americas 2026 (booth 951) during the JETRO Japan delegation visit. It is optimized to do one thing: convert a ten-second booth conversation into a thirty-minute call with Matt Creamer, Eduba's CRO, by naming the orchestration-layer frame for BEP's US market entry and anchoring credibility with a structurally parallel case study (VigilOre) and the Interpretable Context Methodology paper. The page uses Blue Innovation's own color system, typography stack, and polygon-mark visual language so it reads as a document their own marketing team could have produced, with Eduba surfaced only in the footer and at the closing CTA.
